Some Projects I worked on...

FoodFind - Making to discover local restaurants easy and more user interactive

This web application was a creation the started from a love of food. The application aims to connect users with local restaurants more fun. We all know part of the reason people love using tinder is partially because of the noncommitment that goes into it and the way users can easily swipe left with a glimpse of a picture. With the use of Ruby on Rails, and some help from some very useful APIs from Google and Yelp, we have created a site that is dedicated to locating restaurants based on the users location.

Take a look at the repo here or try out the site.

BillPay - Helping users to track, organize and pay bills in one place

Keeping organized isn't easy, and with every day things that tend to build upon each other, it's easy to let things slip your mind. Written with python and utilizing Django's framework, this web application is used to help try to keep bills organized and help to make life a little bit easier. It will keep track on payments and when it will due and give you a notification. It also helps to keep all the websites people need to go to in one easy place.

Check out the project here

ParkIt - An ongoing project to help users determine city parking limits

Everyone knows that looking for parking in San Francisco is almost an impossible feat. When one does find one, there's the endless restrictions that come with public steet parking. This website aims to help people by showing them the parking limits base on the street they are looking at.
